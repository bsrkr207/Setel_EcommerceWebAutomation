# Setel_EcommerceWebAutomation

Maven project with Page Object Model, TestNG, ExtentReports

- Read test data from TestData.Json file (i.e. browser[Firefox/Chrome], website URL's, searchproduct name etc... )
- Generates the execution reports in specified location (currently reports path set to C:\AutomationReports can change the path in TestData.Json file)
  ** based on the mentioned path it will search for the folder and if not exists it will create the folder and store the reports.
